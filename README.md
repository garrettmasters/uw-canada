# Flood Church
2016

## Static HTML site

Getting started
* `$ cd app`
* `$ sudo gem install sass`
* `$ npm install -g gulp`
* `$ npm install`
* `$ gulp`

Site runs at `http://localhost:4000/`


## 2016 WP Theme

Getting started
* `$ cd <theme>`
* `$ sudo gem install sass`
* `$ npm install -g gulp`
* `$ npm install`
* `$ gulp`
